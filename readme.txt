Inspired by [the Susty theme](https://sustywp.com/), I created Pressr. I recognised that few people will use a theme as stripped back as Susty and so set about creating something that would improve any existing site out there - every little helps in reducing a site's size and, therefore, its carbon footprint. So, Pressr will take your current site output and reduce it down and all without affecting how the site looks or feels(1). Simple head to Settings->Pressr and switch on the bits that you want.

Where this is different from other plugins that "optimize" your site is that reducing the page size is the only thing it's concerned about. So, some of the actions that Pressr performs may impact site SEO or performance. Everything can be individually switched via a settings page, however, and any potential impacts are highlighted.

**Note: By default, all Pressr options are switched off, so you can be confident that installing it won't affect your website. Simply head to the settings page and switch on those components that you feel comfortable with**

(1) Okay, not guaranteed, which is why I give you the option to specify which parts of Pressr are used - if anything affects your site negatively, you can switch it off.

Key features include...

* Admin settings screen allows you to precisely define what actions are performed on your site
* WordPress coding standards compliant, including WordPress VIP

Please visit the [Github page](https://github.com/dartiss/pressr "Github") for the latest code development, planned enhancements and known issues.

== Installation ==

Pressr can be found and installed via the Plugin menu within WordPress administration (Plugins -> Add New).

Alternatively, it can be downloaded from WordPress.org and installed manually...

1. Head to Plugins -> Add New -> Upload Plugin and upload the zip file that you downloaded from WordPress.org.
2. Activate the plugin through Plugins -> Installed Plugins.

Whichever way you install Pressr, after activating it, go to Settings -> Pressr to configure.
